Maven pom.xml解析

pom.xml 文件解析

标签的含义

  • modelVersion 指定当前pom的版本
  • groupId实际项目的名称

    公司名称的反写+项目名称

  • artifactId模块的名称

    项目名称+模块名称

  • version 版本号

    三个数字表示 大版本号 分支版本号 小版本号
    0.0.1
    snapshot快照版本
    alpha 内部测试版本
    beta公测版本
    Release稳定版本
    GA正式发布版本

  • packing 打包方式

    默认方式jar

  • dependencies依赖列

    dependency依赖项
    scope依赖范围 对应的maven生命周期
    optional 设置依赖是否可选，决定子项目是否显示依赖此架包
    exclusion 排除依赖传递列表

  • dependencyManagement依赖管理

    不会引用到到实际项目，给子模块继承用

  • build 构建行为

    plugins插件列表
    plugin插件

  • parent用于子模块对父模块的继承
  • modules指定多个模块进行编译
  • name项目描述名
  • ur 项目地址
  • description项目描述
  • developers开发人员信息
  • license许可证信息
  • organization组织信息
